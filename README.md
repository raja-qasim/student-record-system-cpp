# 🎓 Student Record System (C++)

A simple **Student Record Management System** developed in **C++**.
This is a **console-based application** that allows users to manage student records such as adding, viewing, searching, updating, and deleting student information.

This project was created to practice **basic C++ programming concepts including classes, arrays, loops, and input validation.**

---

# 🚀 Features

• Add new student records
• Display all student records in table format
• Search student by roll number
• Update student information
• Delete student records
• Find the topper student (highest marks)
• Input validation for roll number, age, and marks

---

# 🛠 Technologies Used

• C++
• Standard Library (`iostream`, `iomanip`)

# ⚙️ How to Run the Program

### 1️⃣ Clone the repository

```
git clone https://github.com/raja-qasim/student-record-system-cpp.git
```

### 2️⃣ Go to the project folder

```
cd student-record-system-cpp
```

### 3️⃣ Compile the program

```
g++ "Student record system.cpp" -o student
```

### 4️⃣ Run the program

```
./student
```

# 📋 Program Menu

When you run the program, the following menu appears:

```
1. Insert Student
2. Display All Records
3. Search by Roll Number
4. Delete by Roll Number
5. Update Student by Roll Number
6. Find Topper Student
7. Exit
```

Users can choose any option to manage student records.
